# Kiến quốc ký - game chiến thuật mô phỏng

Nội dung game lấy bối cảnh Việt Nam thời kỳ Đổi Mới (1986-2007), các đội đại diện cho các vùng miền Việt Nam cùng nhau xây dựng đất nước.

## Chạy game

- Truy cập [https://kien-quoc-game-2026.web.app/](https://kien-quoc-game-2026.web.app/)
- Hoặc tải app Tauri từ [Release](https://github.com/Keithsel/kien-quoc/releases)

## Dev

- Cài đặt [Bun](https://bun.sh/)
- Cài đặt [Tauri](https://tauri.app/v2/guides/getting-started/installation/)
- Cài các dependencies: `bun install`
- Chạy dev: `bun run dev` hoặc `bun run tauri dev`
- Build: `bun run build` hoặc `bun run tauri build`
